# Interactive Quiz Application
A responsive quiz application built using HTML, CSS, and JavaScript as part of a Frontend Development internship task.

## Description

This project is a category-based quiz application that allows users to test their basic knowledge of HTML, CSS, JavaScript, and Full Stack concepts.
It focuses on clean UI design, responsiveness, and interactive functionality.

## Features

* Category selection
* Rules screen before starting the quiz
* Multiple-choice questions
* Instant feedback for correct and incorrect answers
* Score calculation and result display
* Fully responsive design for desktop and mobile

## Tech Stack

* HTML
* CSS (Flexbox, Grid, Media Queries)
* JavaScript (DOM Manipulation)
